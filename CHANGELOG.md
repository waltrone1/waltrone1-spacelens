# Changelog

All notable changes to **waltrone1 SpaceLens** will be documented in this file.

This project is part of the **WALTRONE Community Tools** collection.

---

## [1.0.0.0] - 2026-05-06

### Added

- Initial public release of waltrone1 SpaceLens
- Windows storage analysis for folders and drives
- Support for local paths and UNC network paths
- Quick drive selection buttons
- Folder tree view with size, percentage, file count, folder count and path
- Top folders overview
- Top files overview
- File category overview
- Category detail view
- Duplicate file detection
- CSV export
- JSON export
- HTML report generation
- Graphical storage overview
- Treemap-style folder usage visualization
- Error summary for permission and access issues
- Modern Windows desktop interface
- Application icon
- PyInstaller / py2exe build files for creating a Windows executable
- GitHub repository structure with README, LICENSE, CHANGELOG, screenshots and documentation folders

### Notes

- waltrone1 SpaceLens is an analysis tool.
- The application does not automatically delete, move or modify files.
- Large drives, protected folders or network paths may take longer to scan.
- Some folders may show access errors if Windows permissions prevent reading them.
